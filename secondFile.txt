Physical Layer (Фізичний рівень):

Опис: Фізичний рівень визначає фізичні аспекти передачі даних через мережу, такі як вид передачі (провідна або бездротова), електричний сигнал, оптичне волокно і т. д.
Приклади пристроїв: Ethernet кабелі, бездротові антени, оптичні кабелі.
Data Link Layer (Рівень зв'язку даними):

Опис: Рівень зв'язку даними відповідає за кадровий доступ до мережі, а також за управління помилками та корекцією на фізичному рівні.
Приклади пристроїв: Мережеві карти Ethernet, комутатори (switches), маршрутизатори (routers).
Network Layer (Рівень мережі):

Опис: Рівень мережі визначає маршрутизацію даних через мережу. Він відповідає за присвоєння IP-адрес, маршрутизацію і розподіл пакетів даних.
Протоколи: IP (Internet Protocol), ICMP (Internet Control Message Protocol).
Приклади пристроїв: Маршрутизатори, L3-комутатори.
Transport Layer (Рівень транспорту):

Опис: Рівень транспорту відповідає за керування з'єднанням між двома пристроями, передачу даних та контроль передачі. Два найпоширеніших протоколи в цьому рівні - TCP і UDP.
Протоколи: TCP (Transmission Control Protocol), UDP (User Datagram Protocol).
Приклади пристроїв: Комп'ютери, сервери.
Application Layer (Рівень застосунків):

Опис: Рівень застосунків містить програми і послуги, які використовують мережу для обміну даними. Це найвищий рівень протокольного стеку і включає в себе численні протоколи для різних послуг, таких як HTTP (для веб-серверів), SMTP (для електронної пошти), FTP (для передачі файлів) і багато інших.
Протоколи: HTTP, SMTP, FTP, DNS (Domain Name System), і багато інших.
